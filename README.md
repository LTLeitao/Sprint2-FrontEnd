# 📸 OptiCam

Projeto front-end de uma aplicação mobile de câmera inteligente com foco em **IA, OCR, filtros e organização de mídia**, desenvolvido com **HTML5 e CSS3**.

O objetivo do projeto foi simular a interface de um aplicativo moderno de câmera/galeria para smartphone, explorando conceitos de **UI Design**, **componentização visual**, **organização de layouts** e **navegação entre telas**.

---

## ✨ Funcionalidades

### 🏠 Home
- Tela inicial do aplicativo
- Cards de acesso rápido
- Navegação entre funcionalidades

### 📷 Câmera Inteligente
- Interface de captura moderna
- Área de foco visual
- Simulação de detecção por IA
- Tags contextuais

### 🖼️ Galeria
- Organização visual das imagens
- Sistema de favoritos ❤️
- Barra de pesquisa
- Navegação entre seções

### 📁 Álbuns
- Organização por categorias
- Seleção de álbuns
- Interface interativa

### ❤️ Favoritos
- Área dedicada para fotos favoritadas
- Remoção/adição de favoritos
- Navegação integrada

### 📝 OCR (Reconhecimento de Texto)
- Simulação de escaneamento OCR
- Extração de texto
- Botões de ação:
  - Traduzir
  - Compartilhar
  - Salvar texto

### 🎨 Filtros
- Preview de filtros
- Seleção visual de efeitos
- Ajustes de:
  - Brilho
  - Contraste
  - Saturação

---

## 🛠️ Tecnologias utilizadas

- **HTML5**
- **CSS3**
- **Font Awesome**
- **Flexbox**
- **CSS Grid**
- **Gradients & UI Effects**

## 🎯 Objetivos do projeto

Este projeto foi desenvolvido para praticar:

- Estruturação de múltiplas páginas
- Organização de CSS em projetos maiores
- Componentização visual
- Criação de interfaces modernas
- Simulação de experiência mobile
- Organização de navegação entre telas
